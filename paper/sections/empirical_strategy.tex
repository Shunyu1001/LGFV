\section{Empirical Strategy}

The baseline empirical analysis estimates whether historical state capacity
predicts the probability of substantive exit. A simple specification is:

\begin{equation}
SubstantiveExit_i = \alpha + \beta HistoricalCapacity_i + X_i'\gamma + \epsilon_i,
\end{equation}

where $SubstantiveExit_i$ indicates whether city-platform case $i$ achieved
substantive exit, $HistoricalCapacity_i$ measures historically rooted state
capacity in the relevant locality, and $X_i$ includes contemporary fiscal,
economic, debt, and political controls.

The first version of $HistoricalCapacity_i$ will be constructed from CHGIS
\citep{chgis2024}. The main measure is Qing administrative density: the number
of late-imperial county or prefectural administrative units intersecting the
contemporary prefecture-level city, scaled by land area. This measure captures
the depth of historical administrative presence rather than contemporary fiscal
strength. The main alternative measure will use CBDB to construct Ming-Qing
elite density, especially the density of high-degree civil service examination
holders by place of affiliation \citep{cbdb2024,bai2016elite}. The first
operational version counts jinshi and juren records by historical place,
assigns those places to contemporary prefecture-level cities using their
coordinates and modern administrative boundaries, and scales the resulting
counts by prefecture land area. This second measure captures local
administrative human capital and elite formation. A third measure, based on
Ming courier routes and stations, can be used as a robustness check for
historical communication and transport infrastructure.

As a pilot validation exercise, I match the current human-validated cases and
three candidate cases to the CBDB-GADM elite-density measure
\citep{cbdb2024,gadm2022}. This exercise does not estimate the main hypothesis,
because the pilot cases were chosen to develop and probe the coding
procedure rather than to form a representative sample. It is nevertheless
useful as a check on whether the historical measure captures substantively
meaningful variation across the kinds of cities that appear in the
source-collection process. Table
\ref{tab:pilot-historical-capacity} and Figure
\ref{fig:pilot-historical-capacity} show the resulting values. Guangzhou and
Hangzhou have high historical elite density, Chengdu and Xi'an occupy an
intermediate position, and the low-capacity cases---Luzhou, Zhuzhou, Zunyi,
and Liupanshui---have much lower values. The key point is not that the pilot
already estimates an effect. Rather, the pilot shows that the coding scheme is
being applied to cases that vary sharply in historical administrative human
capital, including a validated low-capacity nominal-exit case in Zhuzhou and a
near-miss low-capacity functional-persistence case in Luzhou.

A second specification uses exit type as a multinomial outcome:

\begin{equation}
Pr(ExitType_i = k) = f(HistoricalCapacity_i, X_i),
\end{equation}

where $k$ indexes substantive exit, nominal exit, functional transfer, and
liquidation. This specification evaluates whether historical state capacity is
associated with different institutional responses to the same central reform.

\subsection{Inference with surrogate labels}

The large-sample version of the project will use LLM-assisted labels before all
candidate cases can be fully human-validated. Directly using these labels as if
they were true outcomes would create a measurement problem, because LLM errors
may be correlated with source quality, city capacity, document type, or the
visibility of formal compliance language. I therefore treat LLM labels as
imperfect surrogate labels and use human-validated labels as gold-standard
outcomes for inference.

The main inferential strategy will follow the design-based supervised learning
approach proposed by \citet{egami2023dsl}. The procedure has three steps.
First, I draw a validation sample of city-platform cases for human labeling
with known sampling probabilities. This sampling design is part of the
estimator rather than an after-the-fact quality check. Second, I use the LLM
labels, source metadata, and case covariates to predict the gold-standard
labels in the larger candidate sample. Third, I combine the prediction model
with the validation sample through a doubly robust correction, so that the
downstream regression or descriptive estimand is adjusted for surrogate-label
error.

This approach is useful because the quantities of interest are not the
prediction accuracy of the LLM labels. They are downstream relationships, such
as the association between historical state capacity and substantive exit, or
the distribution of exit types across historical-capacity groups. For the
binary specification, the DSL estimator can be applied directly to
$SubstantiveExit_i$. For the multinomial specification, I can construct one
indicator for each exit type and estimate the corresponding type-specific
relationship, with uncertainty that reflects both sampling variation and label
uncertainty. The human-validated label file remains the authoritative source
for final case labels; the DSL procedure provides a way to use the larger
LLM-coded candidate set without treating surrogate labels as error-free.

Mechanism analyses examine whether stronger historical state capacity is
associated with greater formal fiscal substitution after exit. Relevant
outcomes include special-purpose bond use, on-budget infrastructure spending, project
preparation capacity, state asset reorganization, debt restructuring capacity,
and changes in non-platform local SOE debt. These outcomes are not the main
dependent variables of the paper; instead, they help explain why some
localities are able to achieve substantive exit while others rely on responses
such as nominal exit or functional transfer.
