\section{Empirical Evidence}

This section turns the measurement framework into empirical evidence. The
analysis has three aims. First, it shows that official exit is empirically
different from functional fiscal change. Second, it examines whether the
validated labels vary with historically rooted state capacity in the direction
predicted by the theory. Third, it uses the LLM-assisted screening file as an
auxiliary measurement device, while keeping human-validated labels as the
gold-standard outcome.

The evidence is not yet presented as a final population estimate. The
human-validated sample is designed to build and audit the outcome measure. The
LLM surrogate sample is designed to expand coverage under a validation design.
For that reason, the section separates three objects throughout: the
gold-standard labels, the historically matched validated subset, and the
issuer-level surrogate labels used for validation-adjusted descriptive
counts.

\subsection{Gold-standard labels and source validation}

The current gold-standard file contains ninety-four human-validated
city-platform cases. Each label is assigned from original or near-original
source documents rather than from a single exit list or aggregate platform
count. The file records the formal event, the post-event location of the
relevant function, the final label, the plausible alternative label, the source
coverage score, and the rationale for the coding decision. The broader
screening file contains 361 candidate disclosure rows. These rows are useful
for scaling the project, but they are not treated as final outcomes until they
pass the coding rule and human review.

Table \ref{tab:pilot-validation-tiers} summarizes the measurement base used in
the current analysis. Eighty-four of the ninety-four validated labels can
currently be matched to the CBDB-GADM historical-capacity crosswalk. Ten
validated cases remain unmatched because the current crosswalk does not yet
provide a usable city or county match. The empirical tables that compare exit
type to historical capacity use only the matched subset. The full label file
remains the gold-standard outcome file for validation and future sample
expansion.

\input{tables/pilot_validation_tiers}

The validated labels already show why the paper treats LGFV exit as a
measurement problem. The full gold-standard file contains two substantive
exits, eighty-two nominal exits, and ten functional transfers. No liquidation
case has yet entered the gold-standard file. The distribution should not be
read as a representative population share. It is better read as evidence that
formal compliance language commonly coexists with continuing public-project,
fiscal-support, land-development, entrusted-construction, repayment, or
state-asset functions.

The source audit is central to this claim. Appendix Table
\ref{tab:pilot-source-audit} reports the primary source, formal event,
continued-function evidence, and final label for the historically matched
validated cases. The audit trail keeps the coding decision inspectable. A
prospectus may state that new debt does not constitute local government debt,
while other sections of the same document describe infrastructure investment,
fiscal receivables, entrusted construction, or government-linked repayment.
The label is therefore assigned from the joint reading of formal-event evidence
and post-event functional evidence, not from the compliance sentence alone.

\subsection{Descriptive patterns}

Table \ref{tab:pilot-capacity-bin-exit-type} reports the core descriptive
pattern in the historically matched validated sample. The table groups cases
into low, middle, and high historical-capacity bins using the CBDB-GADM
Ming-Qing elite-density measure. The low-capacity bin contains only nominal
exits. The middle bin contains nominal exits and functional transfers. The
high-capacity bin contains all three observed outcomes, including both
substantive-exit cases.

\input{tables/pilot_capacity_bin_exit_type}

This pattern supports a restrained version of the theoretical claim.
Historical state capacity does not mechanically produce substantive exit.
Many high-capacity cases still receive nominal-exit labels. What changes is
the feasible institutional menu. Higher-capacity localities are more likely to
show some departure from pure formal compliance, either by absorbing former
platform functions into fiscal channels or by reorganizing those functions
through a local state-owned system. Lower-capacity cases in the current
validated sample are dominated by formal compliance paired with continuing
platform functions.

Figure \ref{fig:pilot-exit-type-by-capacity-bin} presents the same comparison
as shares. This figure replaces a simple ranking of historical elite density
because the outcome categories, not the capacity measure alone, are the object
of interest. The figure also makes the main empirical point visually clear:
substantive exit is rare in the current validated sample, but when it appears
it appears only in the high-capacity bin; functional transfer appears in the
middle and high bins; nominal exit dominates all bins and fully dominates the
low-capacity bin.

Case comparisons clarify the pattern. Guangzhou City Construction Investment
and Ningbo Development Investment are coded as substantive exit because the
reviewed documents indicate movement away from off-budget government-project
financing. In Guangzhou, former government project work is described as
fiscally funded entrusted management rather than platform financing. In
Ningbo, the reviewed source packet does not show the continuing public project
financing categories that would rule out substantive exit. Xuzhou, Yancheng,
and Hengyang point in the opposite direction. Their documents contain formal
exit or no-government-financing language, but the same issuers continue to
show shantytown redevelopment, entrusted construction, land preparation,
government settlement, fiscal subsidy, or public project repayment functions.

The expanded validated cases prevent a deterministic interpretation. Nanjing
Transportation Construction Investment, Shijiazhuang Transportation
Investment, Haian Tenghai Investment, Gaochun International Cittaslow Cultural
Tourism Investment, and Jiaxing Jiaxiu Development Investment all come from
middle- or high-capacity settings, yet their source packets still support
nominal-exit labels. These cases show that capacity expands the menu of
possible responses rather than guaranteeing substantive exit. The central
empirical distinction is not strong versus weak cities in a simple sense. It
is whether an official exit event changes the institutional location of the
public financing function.

\subsection{Statistical association in the validated sample}

The first statistical test uses the historically matched validated cases. The
dependent variable is institutional change, which equals one for substantive
exit or functional transfer and zero for nominal exit. This binary outcome is
not meant to collapse the theory. Substantive exit and functional transfer are
different institutional fates. The purpose of this specification is to ask
whether historical capacity predicts movement away from the narrowest form of
official compliance, given that the current gold-standard file contains only
two substantive-exit cases.

\input{tables/pilot_lpm_institutional_change}

Table \ref{tab:pilot-lpm-institutional-change} reports linear probability
models with robust standard errors. Across three simple specifications,
historical capacity is positively associated with institutional change. A
one-standard-deviation increase in elite density is associated with a 6.3
percentage point increase in the probability of substantive exit or functional
transfer. Membership in the high historical-capacity bin is associated with a
17.1 percentage point increase relative to other bins. Moving from low to
middle to high capacity is associated with a 13.5 percentage point increase at
each step.

These estimates should be read as descriptive associations, not as final
causal estimates. The sample is a validated measurement sample, and several
important contemporary fiscal controls are not yet fully merged at the case
level. Even so, the models are useful for two reasons. They show that the
theoretical pattern is not driven only by a few named examples, and they define
the specification that the full sample will use once city-level fiscal, debt,
and land-finance controls are complete.

\subsection{Alternative explanations and controls}

The main alternative explanation is contemporary fiscal strength. Wealthier
cities may be able to move platform functions onto budget, replace implicit
debt with formal debt instruments, and maintain investment without relying on
LGFVs, regardless of historical state capacity. A second explanation is debt
pressure. Highly indebted cities may announce exit because central discipline
requires it, while preserving platform functions because fiscal stress makes
substantive exit difficult. A third explanation is land finance dependence:
localities that rely heavily on land development may keep platform functions
alive even after formal exit. Other concerns include province-level policy
differences, bond market disclosure quality, platform administrative rank, and
capital or sub-provincial status.

The full empirical specification therefore uses the following form:

\begin{equation}
\begin{aligned}
InstitutionalChange_i ={}& \alpha + \beta HistoricalCapacity_i \\
&+ X_i'\gamma + \delta_p + \epsilon_i ,
\end{aligned}
\end{equation}

The dependent variable equals one for substantive exit or functional transfer.
Historical capacity is measured with historical administrative or elite
density. The control vector includes contemporary fiscal capacity, debt
pressure, land finance dependence, bond disclosure quality, platform
administrative level, and capital or sub-provincial status. Province fixed
effects are included when the sample supports them. The multinomial version uses
substantive exit, nominal exit, functional transfer, and liquidation as
separate outcomes:

\begin{equation}
Pr(ExitType_i = k) = f(HistoricalCapacity_i, X_i, \delta_p),
\end{equation}

where $k$ indexes the four institutional fates. In the present version, I
report the simpler institutional-change model because the validated sample has
too few substantive-exit cases for a stable multinomial model. The controls
and fixed-effects specification are included here to define the estimating
equation. The present version estimates this equation in two stages. First, it
uses all validated rows with historical-capacity matches and adds the controls
that are already complete in the panel: source or bond-disclosure quality,
capital or sub-provincial status, provisional platform hierarchy, and province
fixed effects. Second, it reports a first-pass full-controls model for the
subset of cities for which GDP per capita, fiscal self-sufficiency, local debt
pressure, and a land-finance proxy have been collected from source-backed
official statistical communiques, budget reports, budget tables, and
public-data compilations. These source-backed controls are useful for moving
the empirical design from placeholder to estimable model, but the
compilation-based entries should still be replaced with official statistical
communiques, fiscal final accounts, debt disclosures, and government-fund
revenue tables before the estimates are treated as final journal evidence.

\subsection{LLM surrogate labels and validation-adjusted inference}

The LLM-assisted screening file expands the measurement process beyond the
gold-standard file. It contains 361 candidate disclosure rows, of which 346
are usable screening observations. Table \ref{tab:llm-screening-status}
separates the broader screening file from the narrower exit-type outcome
sample. The outcome sample contains ninety-four gold-standard labels and 203
LLM surrogate exit-type labels. The remaining usable screening rows include
forty-four source packets for which the model found no direct formal exit or
compliance event and five human-reviewed boundary packets. These rows are
informative for source coverage and attrition, but they are not treated as
exit-type outcomes.

The current surrogate rule is intentionally narrow. It identifies
medium-confidence nominal-exit cases where the source packet pairs formal
compliance language with continuing public-project, land-development,
entrusted-construction, fiscal-support, or state-asset functions. It is not a
general classifier for all four exit types. The 203 disclosure-level
surrogate labels correspond to 158 unique issuers because repeated bond
disclosures often describe the same platform. Table
\ref{tab:surrogate-empirical-flow} converts the disclosure-level screening

\input{tables/surrogate_empirical_flow}

Table \ref{tab:dsl-surrogate-validation} reports the first validation
diagnostic. All sixty-one overlap issuers match the gold-standard label:
nominal exit. I do not interpret this raw precision of 1.000 as literal
certainty. With a Jeffreys correction, the implied precision is 0.992. Using
the Wilson 95 percent lower bound gives a more conservative rate of 0.941.
Applied to the ninety-seven non-overlap surrogate issuers, these estimates
imply between 91.25 and 96.22 expected nominal-exit cases. The result validates
the current surrogate rule as a high-precision screen for nominal exit. It
does not validate unresolved rows as negative outcomes, and it does not
validate the model as a full classifier for substantive exit, functional
transfer, or liquidation.

\input{tables/dsl_adjusted_outcome_distribution}

For inference, I follow the design-based supervised learning logic of
\citet{egami2023dsl}. Let $Y_i$ denote the unobserved gold-standard exit label,
$S_i$ the LLM screening outcome, and $V_i$ an indicator for whether case $i$ is
human validated. In the current implementation, $S_i$ is a one-sided surrogate
for nominal exit. The downstream estimand, such as the association between
historical capacity and institutional change, should therefore be estimated
with a validation-adjusted or doubly robust correction rather than with raw
surrogate labels.

The corrected design has three steps. First, draw a validation sample with
known sampling probabilities, including oversamples of low-capacity and
high-debt localities where source ambiguity is likely to be greatest. Second,
estimate the relationship between surrogate labels and gold labels using
source coverage, document type, historical capacity, debt pressure, land
finance dependence, platform characteristics, and disclosure quality. Third,
estimate the substantive relationship using gold labels where available and
predicted or adjusted outcomes where only surrogate labels are available,
propagating uncertainty from the validation design. The LLM is therefore an
auxiliary measurement instrument. It expands the source review and helps model
labeling error, but it does not replace the human-validated outcome scale.

\subsection{Mechanism evidence}

The mechanism evidence connects the exit labels to the capacities specified in
the theory. Fiscal absorption appears when former platform functions move into
budgetary channels, special-purpose bonds, or fiscally funded project
management. Bureaucratic coordination appears in asset transfers, debt
succession, municipal platform-group reorganization, SASAC entrustment, and
cross-agency implementation documents. Project and state-asset governance
appears when the post-exit arrangement clarifies project preparation,
repayment responsibility, fiscal settlement, asset operation, and the boundary
between state-owned enterprise activity and implicit government financing.

Appendix Table \ref{tab:pilot-mechanism-evidence} reports qualitative
mechanism evidence for the historically matched validated cases. The
substantive-exit cases show evidence that former platform functions were
absorbed into formal fiscal channels or were no longer visible in the reviewed
platform-function categories. The functional-transfer cases show coordination
and reorganization, but the public function remains inside a local
state-owned or platform-like system. The nominal-exit cases tend to preserve
public project, repayment, land-development, entrusted-construction, or fiscal
support functions inside the same issuer after the formal compliance event.

The empirical implication is that the measurement framework is doing more than
renaming official exit. It identifies where the fiscal function goes. In
high-capacity substantive-exit cases, formal exit is paired with fiscal
absorption or credible negative evidence of continued platform financing. In
functional-transfer cases, formal exit is paired with reorganization inside the
local state-owned system. In nominal-exit cases, formal exit is paired with
continued platform-like fiscal functions. This is the institutional variation
that aggregate exit counts obscure.
