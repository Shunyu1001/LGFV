# Candidate Historical-Capacity Prioritization

This memo attaches the CBDB-GADM Ming-Qing elite-density measure to the full candidate city list. The purpose is not to treat the candidate list as a representative sample. It is to discipline the next round of source collection by making historical-capacity variation explicit before additional cases are chosen.

## Coverage

The current candidate file contains 37 platform rows covering 33 unique province-city pairs. All candidate rows match to a GADM ADM2 unit after applying a small name crosswalk for Xi'an and Honghe Hani and Yi Autonomous Prefecture.

The unique cities divide into 11 high-density, 11 middle-density, and 11 low-density cases. The bins are based on the candidate cities themselves, so they should be read as a sampling guide rather than as national cutpoints.

## Recommended Next Collection

The next collection round should keep one high-capacity search, one low-capacity search, and one middle-capacity search active at the same time. This will keep the evidence base from drifting toward only coastal document-rich cases or only debt-stress cases.
