## Coding rationale

The case is coded as nominal exit because formal stripping of government
financing and land-reserve functions coexists with continuing public-project
functions inside the same issuer. A functional-transfer reading is possible if
later evidence shows that fiscal authorities absorbed the financing risk, but
the current packet still places infrastructure and land-related development
functions inside the issuer.
